===============================================
    SOLUCAO PARA ERRO NO OUTRO PC
===============================================

Se aparecer erro como:
"'uardando' nao e reconhecido como comando..."

===============================================
    SOLUCOES:
===============================================

1. EXECUTAR DIRETAMENTE:
   - Clique duas vezes em: Dashboard_KE5Z_Desktop.exe
   - Ignore os scripts .bat se der erro

2. USAR SCRIPT SIMPLES:
   - Clique duas vezes em: EXECUTAR.bat
   - Este script e mais simples e robusto

3. SE AINDA DER ERRO:
   - Clique com botao direito no executavel
   - Selecione "Executar como administrador"

4. VERIFICAR ANTIVIRUS:
   - O Windows Defender pode estar bloqueando
   - Adicione excecao para a pasta do dashboard

===============================================
    ARQUIVOS IMPORTANTES:
===============================================

✅ Dashboard_KE5Z_Desktop.exe - PRINCIPAL (use este)
✅ EXECUTAR.bat - Script simples
✅ ABRIR_DASHBOARD.bat - Script completo
❌ Ignore outros arquivos se der erro

===============================================
    INSTRUCOES RAPIDAS:
===============================================

1. Copie a pasta inteira para o PC
2. Clique duas vezes em Dashboard_KE5Z_Desktop.exe
3. Aguarde carregar
4. Acesse http://localhost:8501 no navegador
